Listas

Lista não ordenada

Usada só para listar elementos sem ser necessária uma ordem específica

  • Elemento 1
  • Elemento 2
  • Elemento 3
  • Elemento 4

Lista ordenada

Usada quando os elementos precisam estar em uma ordem.
Como por exemplo, uma lista de ganhadores de uma corrida

  1. Participante x
  2. Participante y
  3. Participante z
